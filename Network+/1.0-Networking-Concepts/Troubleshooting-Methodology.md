## 📖 Definition
CompTIA uses a standardized 7-step troubleshooting methodology across all exams.

---

## 🔑 The 7 Steps

1. **Identify the problem**
   - Gather information, question users
   - Identify symptoms, check recent changes

2. **Research knowledge base / Internet**

3. **Establish a theory of probable cause**
   - Question the obvious first

4. **Test the theory to determine the cause**
   - If confirmed, determine next steps
   - If not confirmed, establish new theory or escalate

5. **Establish a plan of action and implement**
   - Address root cause not just symptoms

6. **Verify full system functionality**
   - Confirm fix works
   - Implement preventive measures

7. **Document findings, actions, and outcomes**

---

## 🧠 Mnemonic
**"I Really Enjoy Testing Every Verification Document"**

---

## ⚠️ Exam Gotchas
- Step 7 Document is ALWAYS last
- Question the obvious first = check if it is plugged in
- Verify FULL system functionality not just the reported issue
- Always consider recent changes as starting point
