## 📖 Definition

---

## 🔑 Social Engineering

| Attack | Description |
|---|---|
| Phishing | Fake emails to steal credentials |
| Spear Phishing | Targeted at specific person |
| Whaling | Targets executives |
| Vishing | Voice phishing |
| Smishing | SMS phishing |
| Tailgating | Physically following through door |
| Pretexting | Fake scenario to gain trust |
| Shoulder Surfing | Watching someone type |
| Dumpster Diving | Looking through trash for info |

---

## 🔑 Network Attacks

| Attack | Description |
|---|---|
| DoS | Overwhelm system with traffic |
| DDoS | DoS from multiple sources |
| Man-in-the-Middle | Intercept communications |
| ARP Poisoning | Fake ARP replies to redirect traffic |
| DNS Poisoning | Corrupt DNS cache |
| VLAN Hopping | Jump between VLANs |
| Replay Attack | Capture and resend valid data |
| Spoofing | Fake IP/MAC address |

---

## 🔑 Malware Types

| Type | Description |
|---|---|
| Virus | Attaches to files, needs user action |
| Worm | Self-replicates across network |
| Trojan | Looks legitimate, hides malware |
| Ransomware | Encrypts files, demands payment |
| Spyware | Secretly monitors user |
| Adware | Displays unwanted ads |
| Rootkit | Hides deep in OS |
| Keylogger | Records keystrokes |
| Botnet | Network of compromised machines |

## ⚠️ Exam Gotchas
- Ransomware encrypts, does not delete
- Worms spread without user action, viruses need it
- Rootkits are hardest to detect and remove
- ARP poisoning works at Layer 2
