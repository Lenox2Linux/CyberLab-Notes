## 📖 Definition
IEEE 802.11 defines wireless networking standards. Each generation offers improvements in speed, frequency, and range.

---

## 🔑 802.11 Standards

| Standard | Frequency | Max Speed | Key Feature |
|---|---|---|---|
| 802.11a | 5 GHz | 54 Mbps | First 5GHz |
| 802.11b | 2.4 GHz | 11 Mbps | First widely adopted |
| 802.11g | 2.4 GHz | 54 Mbps | Backward compatible b |
| 802.11n Wi-Fi 4 | 2.4/5 GHz | 600 Mbps | MIMO |
| 802.11ac Wi-Fi 5 | 5 GHz | 6.9 Gbps | MU-MIMO beamforming |
| 802.11ax Wi-Fi 6 | 2.4/5/6 GHz | 9.6 Gbps | OFDMA dense environments |

---

## 🔑 Wireless Security Protocols

| Protocol | Security Level | Encryption |
|---|---|---|
| WEP | Broken | RC4 |
| WPA | Weak | TKIP |
| WPA2 | Good | AES/CCMP |
| WPA3 | Best | SAE |

---

## 🔑 2.4 GHz vs 5 GHz

| 2.4 GHz | 5 GHz |
|---|---|
| Longer range | Shorter range |
| More interference | Less interference |
| Fewer channels | More channels |
| Slower | Faster |
| Better wall penetration | Less wall penetration |

---

## ⚠️ Exam Gotchas
- 802.11n = first dual-band standard
- WPA2 uses AES, WPA uses TKIP
- WEP is always the wrong answer for most secure
- Wi-Fi 6 = 802.11ax
- WPA3 uses SAE
