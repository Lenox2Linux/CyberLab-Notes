## 📖 Definition
The OSI (Open Systems Interconnection) model is a 7-layer conceptual framework that standardizes how network communication functions are divided and organized.

---

## 🔑 The 7 Layers

| Layer | Name | Key Protocols/Devices | Data Unit |
|---|---|---|---|
| 7 | Application | HTTP, FTP, DNS, SMTP | Data |
| 6 | Presentation | SSL/TLS, JPEG, encryption | Data |
| 5 | Session | NetBIOS, RPC | Data |
| 4 | Transport | TCP, UDP | Segment/Datagram |
| 3 | Network | IP, ICMP, Router | Packet |
| 2 | Data Link | Ethernet, MAC, Switch | Frame |
| 1 | Physical | Cables, Hubs, NIC | Bits |

---

## 🧠 Mnemonics
- **Top to Bottom (7→1):** "All People Seem To Need Data Processing"
- **Bottom to Top (1→7):** "Please Do Not Throw Sausage Pizza Away"

---

## 🔑 Key Devices by Layer
- Layer 1 — Hub, cables, NIC
- Layer 2 — Switch, bridge, WAP
- Layer 3 — Router, Layer 3 switch
- Layer 4 — Firewall (stateful)
- Layer 7 — Firewall (WAF), proxy

---

## ⚠️ Exam Gotchas
- Switches operate at Layer 2 (Layer 3 switches exist too)
- Routers operate at Layer 3
- Hubs operate at Layer 1
- MAC addresses = Layer 2
- IP addresses = Layer 3
- SSL/TLS = Layer 6 Presentation
